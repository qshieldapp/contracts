# contracts
The prebuilt smart contracts that QShield deploys for you if you choose to have us create and deploy a game contract for you. There are 2 variations for now:  
- 30 place leaderboard: cheaper on the gas fee, but less detailed  
- 50 place leaderboard: costlier on the gas fee, but more elaborate  
- deSci smart contract  
- messenger smart contract  

Please choose the correct setting on our frontend if you do choose the 'Create a Smart Contract for Me' on our App.
